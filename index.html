Minecraft Cutscene Generator

In this page you can create your own cutscenes for minecraft in the form of a data pack.

There are a lot of features to be added to this tool yet.
You'll have to be on spectator mode or flying in creative mode for the cutscene to work correctly.

To install the cutscene, press the download button and save the .zip file in the "datapacks" of your world's folder (or follow this guide).

In order to start the cutscene run the command /function cutscene-[whatever id you chose]:start.
To uninstall the datapack, it is recommended to run the command /function cutscene-[whatever id you chose]:uninstall to delete
the scoreboard objectives you won't be using anymore before deleting the .zip file.
